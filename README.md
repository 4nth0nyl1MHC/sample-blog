# Sample Blog

## Debian or Ubuntu

```
sudo apt install wget zip unzip
cd ~/Downloads
wget https://github.com/arcdetri/blog/archive/master.zip
unzip master.zip
sudo cp -rf blog/html/* /var/www/html/
```

## CentOS

```
sudo yum install wget zip unzip
cd ~/Downloads
wget https://github.com/arcdetri/blog/archive/master.zip
unzip master.zip
sudo cp -rf blog/html/* /var/www/html/
```
